# OmniFlow Chrome Extension

This directory will contain the Chrome Extension responsible for automating
clip processing through the Omni web interface.

## Planned contents

- `manifest.json` — Extension manifest (Manifest V3)
- `background/` — Service worker for background task coordination
- `content/` — Content scripts injected into Omni's web UI
- `popup/` — Extension popup UI
- `assets/` — Icons and static assets
